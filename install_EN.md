# How to Install FALG Extension  
## 1. Download the source  
> [Download Source](https://github.com/wolpis/funny-google-extention/archive/refs/heads/main.zip)  

## 2. Extract the ZIP file  
![image](https://github.com/user-attachments/assets/57f0ab1c-580b-4395-878a-09eb4c9e27ff)  

## 3. Open Chrome Extensions (chrome://extensions/)  
![image](https://github.com/user-attachments/assets/a0eed281-34f7-4468-8a91-0bc3960d98bf)  

## 4. Select the `scripppppppt` folder  
![image](https://github.com/user-attachments/assets/2d72b3ff-5f11-40e3-b920-4f1eab2977fe)  

## 5. The extension has been successfully added!  
![image](https://github.com/user-attachments/assets/783eec2a-c9e3-4487-89c1-f9c10d3dccea)  

## 6. Click the puzzle icon next to the Chrome address bar → Click the three dots next to FALG Extension → Go to "Options"  
![image](https://github.com/user-attachments/assets/f9aed47e-5c8f-46ce-8bef-8025b98a219d)  

## 7. Enjoy the features to your heart’s content!  
![image](https://github.com/user-attachments/assets/831e5a09-1c41-49cf-80d9-d66c733ea3d3)
